inline的空格问题

我我我我 我我我我 我我我我

原因

  • 换行符会被当做空格字符处理

解决方案

  • 使用font-size时，可通过设置font-size:0、letter-spacing、word-spacing解决
  • 使用弹性布局
  • 使用margin负值

浏览器对于空格的默认表现

  • 元素的头尾的空白符会直接忽略
  • 内容中间有多个空格，会被合并成一个空格
